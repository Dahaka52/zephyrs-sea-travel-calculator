**Вот усреднённая база данных по ветровым поясам Земли (идеально подходит для Торилла, т.к. размеры и климат аналогичны).**  
Я опирался на реальные источники:  
- Pilot Charts (NGA Pub 106/107/108 — wind roses с % направлений, Beaufort force, % calms, gales).  
- Copernicus ERA5 + satellite data (месячные wind speed, wave height).  
- Глобальные климатологии (cloud cover, precipitation frequency, thunderstorm reports из reanalysis и ship observations).  

Данные **усреднены для открытого океана** (именно то, что нужно для расчёта скорости парусника).  
Пояса привязаны к широте на вашей карте Торилла (синие стрелки = базовое направление). ITCZ (Doldrums) сдвигается: летом NH — на 5–10° севернее, зимой — южнее.  

**Для Южного полушария** — зеркально (SE пассаты вместо NE, NW westerlies вместо SW и т.д.).  
**Диурнальная вариация** над океаном слабая (в отличие от суши). Днём скорость +0,5–2 узла (сдвиг +5–10% в категории 10+ узлов), ночью — наоборот. В скрипте можно добавлять случайный модификатор ±5–10%.  
**Волнение** — коррелирует с ветром + fetch (открытый океан).  
**Погода** — % вероятности в сутки (сумма может быть >100%, т.к. комбинируется, например облачно + дождь).  

Для удобства скрипта — данные **по сезонам NH** (DJF = зима, MAM = весна, JJA = лето, SON = осень). В коде можно линейно интерполировать по месяцу. Полные месячные таблицы слишком громоздкие, но вариации минимальны (кроме Westerlies и Doldrums).

**Horse Latitudes (~30°)** — переходная зона между пассатами и westerlies: calm ↑ (до 15–20%), ветра слабые и переменные — берите среднее между Trade и Westerlies или отдельно как «calm variant» пассатов.

### Как использовать в скрипте Foundry (рекомендация)
1. Определяй пояс по широте корабля + сезон месяца.
2. Бери % из таблицы → генерируй направление/скорость/волнение/погоду через weighted random.
3. Добавляй случайность ±10% (погода не идеально предсказуема).
4. Для дневного/ночного времени — сдвиг категорий скорости.
5. Корректируй по карте Торилла (если стрелки показывают локальные отклонения — +10–15% к базовому направлению).
